== Description == 

 Image Gallery - allow you to create lightbox image gallery with caption support and ability to add a slideshow or alternate URL.

== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `wp-lightbox-image-gallery` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
